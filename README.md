# Blockchain Regulation in the UK — MSc Dissertation

This repository contains the full data and analysis code for my MSc Business Analytics dissertation:

**Title:**  
Blockchain Regulation in the UK: Text and Sentiment Analysis of Parliamentary Debates and FCA Consultation Papers

## Repository Structure

- **data/**
  - Hansard.xlsx — Real parliamentary debate transcripts collected from Hansard
  - FCA analysis.xlsx — Full FCA consultation papers converted from official PDFs
 
- **notebooks/**
  - Jupyter notebooks for data cleaning, NLP analysis, and visualisation

- **scripts/**
  - Python scripts used for automated data collection and preprocessing

## Data Sources

- Hansard: https://hansard.parliament.uk/
- Financial Conduct Authority: https://www.fca.org.uk/publications/consultation-papers

## Reproducibility

All datasets are derived from publicly available government sources.  
The code provided in this repository enables full reproduction of the analysis presented in the dissertation.
